# Teste de OSF Global Services

## Getting started

### Install

```shell
yarn install
```

### Running local 
This project uses webpack-dev-server to run locally

```
yarn start
```

## Tests

Jest was chosen to run unit tests
```
yarn run test:unit
```

There is a signup integration test made with Cypress
```
yarn run test:e2e
```
